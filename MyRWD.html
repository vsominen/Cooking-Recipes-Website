Cooking Recipe Website

  • A website is developed using HTML, CSS, Javascript containing the recipes with the procedure, images and videos. Bootstrap is used for adding various styles for the website.
  • A search bar is implemented using JS and PHP for searching the recipes with their names, cuisines or with the type of food like vegan, desserts, cakes, etc.
  • A dropdown button is implemented using Javascript, for selecting the meal type like breakfast, lunch and when they are selected, the results are retrieved from the database.
  • Users can also SignUp to the website for getting notification of new recipes and, they can provide feedback and rating for the recipe.

Our Purpose

To provide best recipes with detailed instructions, ingredients, nutrition values along with videos. So that everyone can make delicious and healthy food at home.
